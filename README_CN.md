# GitHubFollow
Gitstar自动加粉工具
# 快速入门
## Linux
### 第一步
安装python2.x

安装pip, 从 这里下载[get-pip.py](https://bootstrap.pypa.io/get-pip.py)

接着运行:

> python get-pip.py

安装依赖

> pip install -r requirements.txt
### Step2
打开```main.py```,替换以下内容  
```
#-----------------------------
#GitStar
NAME="w568w"
PASSWORD="xxxx"
#GitHub
GITNAME="w568w"
GITPASSWORD="xxxx"
#-----------------------------
```
把它们改成你自己的信息。  
### Step3
运行```python -u main.py```,  
完事儿！
